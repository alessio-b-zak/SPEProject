GeoJsonParser

class GeoJsonParser

Parses a JSONObject and places data into their appropriate GeoJsonFeature objects. Returns an array of GeoJsonFeature objects parsed from the GeoJSON file. Based on googlemaps :

https://github.com/googlemaps/android-maps-utils/blob/master/library/src/com/google/maps/ android/data/geojson/GeoJsonParser.java

Apache License 2.0

Functions

parseFeature static fun parseFeature(): GeoJsonFeature

Parses a single GeoJSON feature which contains a geometry and properties member both of which can be null. Also parses the bounding box and id members of the feature if they exist.
parseGeometry static fun parseGeometry(): Geometry

Parses a single GeoJSON geometry object containing a coordinates array or a geometries array if it has type GeometryCollection
